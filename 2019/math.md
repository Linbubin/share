## 点到线的距离
```
点到直线的距离可以用公式：|AX+By+C|除以A的平方加B的平方的算数平方根
举个例子：求P（5，3）到直线4X-3Y+1=0的距离
解：L=|4*5-3*3+1|除以（16+9)的算数平方根=2.4
```

### 两个点的直线方程式
```
(x1,y1) (x2,y2)

直线方程两点式:
(y-y1)/(y2-y1)=(x-x1)/(x2-x1)
y-y1=(y2-y1)(x-x1)/(x2-x1)
y=(y2-y1)(x-x1)/(x2-x1)+y1
即y=x(y2-y1)/(x2-x1)+(x2y1-x1y2)/(x2-x1)
```
